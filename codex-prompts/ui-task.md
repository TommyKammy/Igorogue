# Codex Prompt — UI TASK

指定UI Screen SpecとRules Canonを読み、モックアップの視覚を盲目的にコピーせず、情報優先度と状態を実装してください。

## 必須状態

- default
- hover/focus
- selected
- illegal target
- immediate capture
- self-king danger
- counterattack imminent
- disabled
- gamepad focus

## 検証

- 1280×800と1920×1080
- 文字サイズ設定
- 色覚に依存しない表示
- キーボード・ゲームパッド操作
- 盤面イベントとUI数値の順序一致
