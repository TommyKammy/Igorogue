# v0.2.5 Design Patch — Global Momentum Gate

## Added

- Accepted版[[FEAT-002 Momentum]]。
- 全流儀共通の余勢resource lifecycle。
- 黒領地成立による普遍生成source。
- 地合い流の`facility_build_grants_momentum`追加source。
- `momentum_reach`のdistance 2、orthogonal、empty midpoint契約。
- 前線前進ドローの最短王石距離と解決後収入gate。
- MOM-01〜MOM-19の仕様fixtureと専用checker。

## Changed

- Rules Canonを0.2.5へ更新。
- 「新領地または新施設で全流儀が余勢生成」という曖昧記述を廃止。
- 通常配置が合法な点では余勢を使わないことを明記。
- 余勢をターン終了で失わず、戦闘境界でresetすることを明記。
- FEAT-001、Styles、system data、Domain、Event、UI、Telemetryを同期。

## Deferred

- A-6の全流儀rule同期。
- A-7反攻数値再設計。
- 製品Rules Kernelへの実装。
