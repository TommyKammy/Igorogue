# v0.2.3 Design Patch — Facility Intersection Semantics

## Purpose

Fable 5レビューのP0 A-3を解消し、施設点についてRules Kernel、領地、呼吸点、敵侵入、UIが別々の解釈を持つことを防ぐ。

## Accepted decision

- [[ADR-0012 Facility Sites Are Empty Intersections]]
- 施設はStone layerと別の空点上マーカー
- 施設点は呼吸点、空点領域、領地サイズ、配置候補で空点
- 合法な石配置で直接破壊
- 領地喪失では停止、奪還で再稼働
- 容量は新規建設ゲート。分割後over-capacityは既存稼働を維持
