# v0.2.4 Design Patch — Coordinates and Initial Symmetry

## Purpose

Fable 5レビューのP0 A-4を解消し、表示座標、内部配列、盤面図、敵タイブレーク、`StoneTopologyKey`、初期配置が別々の方向・原点を暗黙採用することを防ぐ。

## Accepted contract

- [[Coordinate System and Initial Position]]
- CanonicalPointは`(x,y)`、各軸1〜7
- `x`は左→右、`y`は下→上
- 盤面図は上から`y=7`、下が`y=1`
- InternalPointは0〜6、外部データは1-based CanonicalPoint
- Canonical point orderは`y`昇順、次に`x`昇順
- 点対称は`reflect(x,y)=(8-x,8-y)`
- 標準初期配置`standard_v0_2`は色・役割交換付き点対称
- 各初期王石グループは3石連結・実呼吸点7
