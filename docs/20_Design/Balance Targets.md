# Balance Targets

正式盤面実装後に置換する仮説値。

| 指標 | 目標 |
|---|---:|
| 熟練者の中規模成長 | 90〜100% |
| 熟練者の真の爆発 | 45〜60% |
| 第2戦〜第3戦序盤までの爆発 | 15〜25% |
| 爆発の事前仕込み | 85〜95% |
| 固定構成の爆発率 | 80%未満 |
| 爆発なし勝利 | 25〜40% |
| 熟練者の爆発なし敗北 | 8〜15% |
| 単一ビルドの勝利占有 | 25%未満 |
| 3印が最適な環境 | 40%未満 |
| 爆発ラン中の反攻遭遇 | 60%以上 |
| 反攻予告後の守備選択 | 20%以上 |
| 反攻予告後の攻撃継続 | 20%以上 |

## 警告条件

- 一つの触媒が適格報酬の60%以上で選ばれる。
- 報酬スキップがほぼ常に正解／不正解。
- 高コミが棋譜片稼ぎの最高効率。
- 施設を最大まで積むことが毎戦の定石。
- 爆発後に守る選択が存在しない。
