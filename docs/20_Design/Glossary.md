# Glossary

| 用語 | 意味 |
|---|---|
| CanonicalPoint | 外部保存・表示用の1-based座標`(x,y)`。左下`(1,1)`、右上`(7,7)` |
| InternalPoint | Rules Kernel配列用の0-based座標`(ix,iy)`。外部データへ露出させない |
| Canonical point order | `y`昇順、次に`x`昇順の安定順。index 0は`(1,1)` |
| 点対称 | 中央`(4,4)`を中心とする`reflect(x,y)=(8-x,8-y)` |
| 王石 | 捕獲されると勝敗が決まる石 |
| グループ | 上下左右で連結した同色石集合 |
| 実呼吸点 | グループに隣接する重複なし空点 |
| 仮呼吸点 | 石instanceへanchorする期限付き耐久。effect instanceごとに加算し、敵ターン終了掃引で一括失効する |
| Continuous liberty modifier | 霊泉等、条件を満たす間だけ再導出され、timed失効掃引の対象にならない呼吸点補正 |
| 仮呼吸点失効掃引 | due effectを一括失効し、同一snapshotの有効呼吸点0グループを同時captureする敵ターン終了phase |
| アタリ | 有効呼吸点1 |
| 前線 | 自石に隣接する空点 |
| 終着 | 即捕獲する最後の呼吸点への例外配置 |
| 盤面反復禁止 | 戦闘中に一度現れた石色・王石配置へ戻る石変化を禁止する規則 |
| StoneTopologyKey | 49交点の空／黒／白／黒王石／白王石だけを並べた反復判定キー |
| 領地 | 単色石だけに接する空点領域。施設点も通常の空点として含む |
| 施設 | Stone layerを占有しない空点上の生産マーカー。合法な石が同点へ置かれると破壊 |
| 余勢 | 全流儀共通の戦闘資源。黒領地成立で生成し、eligibleな前線札を上下左右へ1交点遠く配置する |
| Momentum reach | 余勢1を使う代替配置mode。黒石から上下左右へ距離2、中間Stone layerが空の追加候補 |
| 妙手倍率 | 一ターンの盤面成果を束ねる倍率 |
| 家元印 | 永続解放するラン開始時持込能力 |
| コミ | 家元印に対する敵側補償値 |
| 厚みトークン | 白王石が一度耐える可視防御 |
| 反攻 | 100表示点到達後、次敵ターンの通常行動後に行う予告済み追加敵行動 |
| Pending | 次の敵ターンに反攻追加行動1回が予約済みの状態 |
| 熱 | 全コミ帯で妙手倍率が各ターン初めてx3へ達した時、反攻を進める値 |
| 真の爆発 | 分析上の急加速分類。プレイヤールールではない |
