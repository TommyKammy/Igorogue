# Rewards Shop and Run Economy

## カード報酬

- 3択、取得義務なし。
- 現在ビルド約50%、流儀約25%、ワイルド約25%の重み。
- 2回連続で関連候補なしの場合、次回1枠だけ救済。

## 小ショップ

第1戦後に遺物候補2、カード・削除等を提示。狙った触媒は保証しないが、序盤から壊れるランの機会を作る。

## 魂

ラン内通貨。捕獲、戦闘報酬、祭壇、妙手効果から得る。攻撃型が経済を独占しないよう捕獲基本報酬に上限を持つ。

## 棋譜片

永続解放通貨。初回発見、ボス、段位、実績を中心に固定量。コミ・妙手倍率で増やさない。

## 棋力スコア

消費不可。コミ、妙手、速度、残存安全度で増幅し、ランキングと自己記録へ使う。
