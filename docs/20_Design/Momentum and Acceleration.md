# Momentum and Acceleration

詳細正本は[[FEAT-002 Momentum]]。

## 余勢の役割

経済力を気へ加算するだけでは、手札と盤面アクセスが不足して加速が止まる。
余勢は、領地の成立を前線アクセスへ変換する全流儀共通の戦闘資源である。
特定領地、施設、流儀へ紐付かず、戦闘中はターンをまたいで最大2まで保持する。

## 全流儀共通の生成

黒側の一つのatomic resolutionで、少なくとも一つの点が非黒領地から黒領地へ変わると余勢+1。
同じ解決で複数領地が成立しても+1だけである。

## 地合い流の追加生成

地合い流だけは、黒施設の合法な新規建設でも余勢+1。
他流儀も施設を利用できるが、施設建設だけでは余勢を得ない。

## 消費

`frontline`を印刷タグに持つ単石配置札は、余勢1を使い`momentum_reach`を選べる。

```text
黒石 S
空点 M
対象 T
```

`S`から上下左右へちょうど2、途中`M`に石がなく、`T`が通常配置では合法でない場合に限り、`T`を追加候補にする。
通常合法点では余勢を消費しない。

## 前線前進ドロー

余勢配置後に黒石群から白王石までの最短Manhattan距離が縮まり、解決後の予測黒領地収入が4以上なら、各プレイヤーターン1回だけ1ドローする。

## 爆発の構造

```text
黒領地成立
→ 余勢+1
→ 一歩遠い前線札
→ 領地または捕獲
→ 余勢再生成／気回収
→ 前線前進ドロー
→ 触媒
→ 妙手倍率と反攻
```

どの札に余勢を使うかは任意であり、自動発動しない。
