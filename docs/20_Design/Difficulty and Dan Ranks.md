# Difficulty and Dan Ranks

段位は全解放後の長期目標。単純な数値増加より、新しい制約を追加する。

| 段位 | 例 |
|---|---|
| 入門 | 捕獲予告を強く表示、反攻を1ターン遅延 |
| 5級 | 敵護衛配置が変化 |
| 1級 | 全戦闘に反攻ゲージ |
| 初段 | ワイルド報酬に呪い候補 |
| 二段 | ボス盤面条件2つ |
| 三段 | 施設破壊点が1ターン封鎖 |
| 名人 | 白王石が厚みトークン1で開始 |

流儀別に最高段位を記録する。
