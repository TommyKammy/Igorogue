# Core Loop and Run Structure

## 戦闘ループ

1. 現在の領地を再計算。
2. 基礎気3と領地収入を得る。
3. 手札を引く。
4. 敵意図、捕獲予告、反攻予告を確認。
5. カードを使用。
6. 各効果ごとに捕獲、トリガー、領地、施設、妙手を解決。
7. 敵行動と反攻を解決。
8. 王石捕獲またはターン制限まで繰り返す。

## 4戦ActのVertical Slice

```text
通常戦
→ カード報酬＋小ショップ
→ 通常戦
→ 休憩またはイベント
→ エリート
→ ボス
```

## 1ランの製品版仮説

2 Act、8戦前後、30〜45分。

## 発火曲線

- 第1戦：小領地、基礎カードの役割確認。
- 第2戦：最初の触媒または施設シナジー。
- 第3戦：早期爆発が起こり得る。
- ボス：完成したエンジンと反攻圧力の決着。

毎ランの真の爆発は保証しない。代わりに、中規模成長はほぼ毎ラン感じられることを目標にする。
