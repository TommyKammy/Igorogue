# Game Design Overview

## ジャンル

囲碁ベースの盤面型デッキ構築ローグライト。

## 戦闘の目的

敵の白王石を含むグループの有効呼吸点を0にして捕獲する。自分の黒王石グループが捕獲されるか、ターン制限を超えると敗北する。

## 中心ループ

```text
石を置く
→ グループ・呼吸点・捕獲を解決
→ 領地を作る
→ 領地から気を得る
→ 施設・余勢・ドローへ変換
→ より多く／遠くへ打つ
→ 敵王石を包囲
→ 爆発後の反攻前に決着
```

## 三つの時間軸

### 一手

合法点、捕獲、アタリ、領地完成を読む。

### 一戦

経済を育てるか、速攻するか、反攻を受けるかを判断する。

### 一ラン

報酬、遺物、カード削除で、流儀の初期方針を別のエンジンへ変える。

## プレイヤーが管理するもの

- 気
- 手札と山札
- 自王石の安全
- 領地収入と施設容量
- 余勢
- 妙手倍率
- 反攻ゲージ
- 魂
- 遺物枠
