# Project Brief

## 仮題

**Igorogue（イゴローグ／仮）**

## 一文説明

手札から石と手筋を打ち、囲った領地を経済エンジンへ変え、敵王石を捕獲するデッキ構築ローグライト。

## 中心的な差別化

- エネルギー増加が盤面上の領地に見える。
- カードと遺物が囲碁ルールの価値を反転させる。
- 仕込みから発火し、短時間で急加速する。
- 熟練は偶然を消すのではなく、与えられた機会を爆発へ変換する能力として表現する。
- 永続解放は主に選択肢を増やし、持込強化はコミで敵に補償する。

## プラットフォーム

PCを第一対象とし、マウス・キーボードとゲームパッドを想定する。Steam Deck相当の小画面を将来の確認対象に含める。

## 初期製品仮説

- 1ラン30〜45分
- 7×7盤
- 2 Act、8戦闘前後
- 4つ以上の流儀
- 最大5遺物、最大3家元印
- 2Dドット絵風
