# Metrics Dictionary

| 指標 | 定義 |
|---|---|
| medium_growth | 収入、エンジン、ターン出力が基準を超えたラン |
| true_explosion | 絶対出力、直近比、盤面イベントを満たす急加速 |
| early_explosion | 第2戦まで、または第3戦序盤 |
| setup_explosion | 直前3ターンに2以上の仕込みイベント |
| no_explosion_win | 真の爆発なしの完走 |
| no_explosion_loss | 真の爆発なしの敗北 |
| build_share | 勝利ランに占めるタグ構成比率 |
| seal_slot_optimum | 環境ごとの最適家元印枠数 |
| comprehension | 敗因・次行動を正しく説明できた割合 |

classifierはバージョン管理し、定義変更前後を直接比較しない。
