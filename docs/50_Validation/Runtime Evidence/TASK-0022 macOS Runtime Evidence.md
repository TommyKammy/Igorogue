- adds the Godot-local solution and generated C# UID required by Godot 4.7;
- verifies the native executable, managed data directory, and `Igorogue.Godot.dll`;
- archives the complete Windows export instead of the executable alone;
- separates Godot platform-specific restore locks from committed development locks;
- fails wrappers and CI if export rewrites a committed lock file.
