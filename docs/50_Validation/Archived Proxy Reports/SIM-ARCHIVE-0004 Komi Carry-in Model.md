# SIM-ARCHIVE-0004 Komi Carry-in Model

## 比較

恒久強化、王石防御のみ、段階式コミ、敵強化選択、個別カウンター、滑らかな直交型コミを比較。

## 発見

- コミなしでは全解放時に最強3印が固定化。
- 段階式はコミ4と5の崖を作る。
- 個別カウンターは解放の喜びを打ち消す。
- 白王石厚み＋反攻速度へ滑らかに変換する方式が最良。

## 報告値

最終候補では最適枠数平均約1.9、3枠最適環境約28%、最適コミ平均約4.2と報告された。


> [!warning] 再現性
> 元コード、seed、生データが現存しないため、正式な証拠として使用しない。
