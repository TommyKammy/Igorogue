# SIM-ARCHIVE-0003 v0.2 System Comparisons

## 主な発見

- 遠隔配置札を安くすると領地を無視した王石暗殺へ戻る。
- 施設型は遅いが高収入。侵入敵が必要。
- 捨て石は敵依存だけでは発火しない。
- 種石コスト2は遅すぎ、コスト1候補が有望。
- 完全ランダム報酬は成立しないデッキを増やした。
- 流儀関連候補を一部保証する方式がビルド分散を改善した。
