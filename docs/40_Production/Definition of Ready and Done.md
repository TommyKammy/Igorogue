# Definition of Ready and Done

## Ready

- 一文の成果。
- 非対象。
- 正本リンク。
- 受け入れ条件。
- 変更可能領域。
- 検証方法。
- 依存完了。

## Implemented

コードとデータが動作し、TASK内テストが通る。

## Tested

関連回帰、golden replay、lint、ビルドが通る。

## Validated

シミュレーションまたは人間プレイテストで、設計上の仮説を確認した。

Doneはタスク種別によりImplementedまたはTestedまで。Featureの「面白さ完了」はValidatedを必要とする。
