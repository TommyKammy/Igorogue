# M3 Acceleration Lab

## Purpose

コンテンツ量産前に、Igorogue固有の「仕込み → 発火 → 反攻」が本当に面白いかを検証する最重要ゲート。

## Minimal content

- 流儀2
- カード12〜16
- 遺物4
- 施設2〜3
- 余勢
- 触媒1〜2系統
- 妙手倍率
- 反攻と熱
- 敵2種

## Exit criteria

- 熟練テスターが発火条件を言語化できる。
- 固定構成で爆発が保証されない。
- 爆発しない勝利も成立する。
- 爆発直後に「取り切る／守る」の判断がある。
- 爆発ラン中の反攻遭遇が60%以上。
- 反攻予告後に攻撃継続と守備使用の両方が観測される。
- 施設を積むだけの定石にならない。
- 盤面を読まずカードだけ回す構成が支配しない。

## Stop condition

このゲートを満たさない限り、v0.2のカード・敵・アートを量産しない。
