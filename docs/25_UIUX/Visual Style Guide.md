# Visual Style Guide

## 方向性

**2Dドット絵 × 夜の棋院 × 金継ぎ × 墨と木。**

Balatro的な「少数要素が強い視覚リズムを作る」考え方を参照するが、レトロカジノではなく、夜の寺社・棋院をモチーフにする。

## 基準解像度

480×270の論理解像度を4倍して1920×1080へ表示する案。ピクセルパーフェクトスケーリングを基本とし、UI文字は高解像度テキスト層を別途検討する。

## 形状

- 1〜2pxの硬い枠。
- 角丸を多用せず、木札・金属枠・紙札の直線を使う。
- 選択状態は発光だけでなく二重枠と小さな動きで示す。
- 色だけに依存せず、形とアイコンを併用する。

## 演出

- 捕獲：石が砕けるのではなく、墨へほどけて碁笥へ吸われる。
- 領地完成：境界を墨線が走り、内部に薄い紋様。
- 妙手：朱印スタンプ。
- 反攻：白い鐘音と赤い盤縁パルス。
