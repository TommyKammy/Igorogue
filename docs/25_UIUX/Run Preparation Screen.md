# Run Preparation Screen

![[mockup_02_run_preparation.png]]

## 目的

ボス情報を見て、流儀1つと家元印0〜3個を選ぶ。強さ選択ではなく、開始方針とリスクを選ぶ画面にする。

## 必須表示

- Act 1ボス、固有行動、盤面条件。
- 流儀の長所と明確な制約。
- 家元印の効果とコミ。
- 合計コミ。
- 白王石厚みの概算。
- 最初の反攻予測。
- 棋力スコア倍率。

コミの数値だけでなく、「短期決着向け」「長期戦危険」などの言語要約を出す。
