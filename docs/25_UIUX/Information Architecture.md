# Information Architecture

```text
Title
├─ New Run
│  ├─ Boss Preview
│  ├─ Style Select
│  ├─ Family Seal Loadout
│  └─ Run
│     ├─ Act Map
│     ├─ Battle
│     ├─ Reward
│     ├─ Shop / Rest / Event
│     └─ Run Result
├─ Kiin / Meta Hub
│  ├─ Unlock Board
│  ├─ Seal Box
│  ├─ Kifu Shelf
│  ├─ Tsumego Room
│  └─ Dan Challenge
├─ Collection
└─ Settings
```

戦闘画面から一手で開ける詳細は、デッキ、遺物、用語、ログに限定する。メタ進行画面へは戦闘中に遷移しない。
