# UI UX Overview

## 目標

囲碁未経験者でも「今危険な石」「次に取れる石」「この手で増える経済」を一目で理解できる。一方で、熟練者向けには情報を隠さず、予測差分を高速確認できる。

## 情報優先度

1. 王石の即時危険
2. 敵の次行動と反攻
3. 選択カードの合法点・結果
4. 気、手札、余勢
5. 領地収入と施設
6. 妙手倍率とスコア
7. 山札・捨て札等の詳細
