# Interaction and Input

## マウス

カード選択 → 合法点ホバー → 差分確認 → クリック確定。ドラッグを必須にしない。

## ゲームパッド

盤面カーソルと手札フォーカスを明確に分け、ワンボタンで切替。合法点間のスナップ移動を用意する。

## Undo

未確定の対象選択は戻せる。コマンド確定後は原則戻せない。設定で「危険手確認」を有効にできる。

## 危険確認

次の手で自王石グループが敵の即捕獲候補になる場合だけ、二度押し確認を出す。毎手の確認はしない。
