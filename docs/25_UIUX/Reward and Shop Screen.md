# Reward and Shop Screen

![[mockup_03_reward_shop.png]]

## カード報酬

3択の下に、候補生成の関係ラベルを表示する。

- 流儀シナジー
- 現在ビルド
- ワイルド

これは強さ保証ではなく、なぜ候補へ出たかの説明である。

## デッキ診断

仕込み、発火、攻め、守りを粗い4指標で表示する。ただし「推奨カードを自動選択」せず、現在の偏りを示すだけにする。

## ショップ

遺物候補2、カード、削除、再抽選など。購入義務なし。価格と枠交換を同時に確認できる。
